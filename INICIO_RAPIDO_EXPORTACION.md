## 📥 ¿Cómo Descargar Datos?

### **Pacientes**
```
1. Módulo → Pacientes
2. Botón "Exportar" (esquina superior derecha)
   ├─ Listado Pacientes (PDF)  → Todos los pacientes
   ├─ Listado Pacientes (Excel) → Tabla completa
3. O en cada fila: ícono 📥 → Descarga Carpeta Médica (PDF)
```

### **Asignaciones**
```
1. Módulo → Asignaciones
2. Botón "Exportar" (esquina superior derecha)
   ├─ Listado Asignaciones (PDF)
   ├─ Listado Asignaciones (Excel)
```

### **Citas**
```
1. Módulo → Citas
2. Botón "Exportar" (en la parte superior)
   ├─ Listado Citas (PDF)
   ├─ Listado Citas (Excel)
```

### **Tratamientos**
```
1. Módulo → Tratamientos
2. Botón "Exportar" (en la parte superior)
   ├─ Listado Tratamientos (PDF)
   ├─ Listado Tratamientos (Excel)
```

## 📊 ¿Qué incluye cada exportación?

### PDF
- Encabezado con nombre de la clínica
- Tabla profesional con bordes

- Información completa

### Excel
- Múltiples columnas
- Estilos y colores
- Ancho automático de columnas
- Fácil de filtrar y buscar

## 🎯 Funciones Especiales

### **Carpeta Médica Completa**
Descarga la carpeta médica de un paciente con:
- Datos personales
- Antecedentes familiares
- Antecedentes personales
- Antecedentes no patológicos
- Antecedentes ginecológicos (si aplica)

Disponible en PDF (una página bonita) o Excel (múltiples hojas)

## 🔧 Si algo no funciona

1. **Revisa la consola** (F12 → Console)
2. **Asegúrate de tener datos** en el módulo
3. **Intenta en otro navegador** (Chrome es mejor)
4. **Vacía el caché** (Ctrl+Shift+Del)
